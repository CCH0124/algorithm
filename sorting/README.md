## Heap Sort


Heap Sort 通常會把陣列想像成一棵完全二元樹（Complete Binary Tree）。而一個合格的「最大堆疊」，必須滿足一個鐵律*任何一個節點（爸爸）的值，都必須大於或等於它的左右子節點（兒子）*。整棵樹的最大值，一定會乖乖坐在最頂端（根節點 Root）。

雖然在腦海中把它畫成一棵樹，但它在記憶體中其實就是一個普通的一維陣列。透過簡單的數學公式，我們可以直接用索引（Index）找到親屬關係：

* 如果一個爸爸的索引是 `i`
    * 它的左兒子位置就在：`2 * i + 1`
    * 它的右兒子位置就在：`2 * i + 2`
    * 子節點i的父節點在位置 $$\lfloor (i-1)/2\rfloor$$
